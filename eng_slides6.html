QUARTO

QUARTO

  • quarto manuscript is design for scientific document authoring, with emphasis in reproducibility, since it encourages the readers to explore processing scripts (notebooks).

  • Offers multiple static outputs (.pdf, .docx) that follow an specific template from a magazine.

  • quarto publish allows to deploy the website in Quarto Pub or GitHub Pages .

  • In combination with targets, plots/tables/maps are regenerated with new data input, or script modification, updating the results.
